## Static Website Template

A template for creating static sites.

**Languages/Libraries:**

* [NodeJS][nodejs]
* [Metalsmith][metalsmith]
* [Bootstrap][bootstrap]
* [Knockout][knockout]
* [Coffeescript][coffeescript]
* [HAML][haml]
* [SASS][sass]

## Usage

## Setup

`cake setup`

## Build

Single: `cake build`  
Watching: `bundle exec guard`
> Run `gem install bundler && bundle install` to setup Bundler/Guard
